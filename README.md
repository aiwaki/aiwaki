# aiwaki

I build practical tools for routing, automation, and cross-platform systems.

Current focus: [Slipstream](https://github.com/aiwaki/slipstream) — smart routing for blocked and degraded services in Russian networks.

## Projects

- [Slipstream](https://github.com/aiwaki/slipstream) — smart routing for blocked and degraded services in Russian networks.
- [makc](https://github.com/aiwaki/makc) — no-cgo mouse and keyboard control for Go on Windows, macOS, and Linux.
- [bluetrack](https://github.com/aiwaki/bluetrack) — Android Bluetooth/USB HID trackpad and input-control experiments.
- [gleam-browser-extension](https://github.com/aiwaki/gleam-browser-extension) — a template for Gleam browser extensions.
